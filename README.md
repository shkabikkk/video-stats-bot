# 🎥 Video Stats Bot

Telegram-бот для аналитики статистики видео.  
Понимает вопросы на русском языке, преобразует их в SQL с помощью GigaChat и возвращает точный числовой ответ.

---

## 🚀 Возможности

- 📊 Подсчет количества видео и креаторов  
- 📅 Фильтрация по датам (день, месяц, диапазон)  
- 📈 Агрегация данных (суммы, средние, максимумы)  
- 👤 Детальные запросы по конкретным креаторам  
- 📆 Анализ динамики просмотров по дням  
- 🔢 Ответ всегда одним числом  

---

## ⚡ Быстрый старт

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/shkabikkk/video-stats-bot.git
cd video-stats-bot
```

### 2️⃣ Запустить одной командой

```bash
docker-compose up --build
```

✅ При первом запуске автоматически:

- создастся база данных  
- создадутся таблицы  
- загрузятся данные из data/videos.json  
- применятся все переменные из .env  

---

## 💬 Проверка работы

Напишите боту в Telegram. Примеры запросов:

- «Сколько всего видео?»
- «Сколько видео набрало больше 1000 просмотров?»  
- «На сколько просмотров выросли все видео 28 ноября 2025?»  
- «Сколько разных видео получали новые просмотры 27 ноября 2025?»  

Бот всегда отвечает одним числом.

---

## 🔐 Переменные окружения

Все необходимые токены находятся в файле .env:

- BOT_TOKEN — токен Telegram-бота  
- GIGACHAT_CREDENTIALS — base64 ключ для GigaChat  
- GIGACHAT_SCOPE — область доступа  
- DB_HOST, DB_PORT, DB_USER, DB_PASSWORD, DB_NAME — настройки PostgreSQL

## 🧩 Устранение проблем

### Бот не отвечает

```bash
docker-compose logs bot
```

### Ошибки с GigaChat

- Проверьте правильность GIGACHAT_CREDENTIALS в .env  
- Убедитесь, что на счету есть токены  

### Данные не загрузились

```bash
docker-compose exec bot python load_data.py
```
